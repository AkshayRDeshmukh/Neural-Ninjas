# 🥷 Neural Ninjas

### *Play. Experiment. Become an AI Ninja.*

> ⚡ Welcome to **Neural Ninjas** — a fun, interactive playground where kids (and curious minds) learn AI by *doing*, not just reading.

---

## 🎯 What is Neural Ninjas?

**Neural Ninjas** is a collection of playful AI experiments powered by modern models like:

* 🤚 Hand Tracking
* 😊 Face Detection
* 🎙️ Voice Interaction
* 🧠 Smart AI Logic

All designed to make learning **visual, interactive, and FUN**.

---

## 🚀 Why this exists

Because learning AI shouldn’t feel like studying a textbook 😴

It should feel like:

* 🎮 Playing a game
* 🎨 Creating something cool
* 🤯 Seeing AI react in real-time

---

## 🧪 What you'll find here

Each experiment is like a **mini mission** 🥷

* ✋ Control visuals with your hands
* 👀 Detect faces and expressions
* 🎤 Interact using voice
* 🎯 Solve fun AI challenges

> Example: *Move your hands → watch the neural web react in real-time*
